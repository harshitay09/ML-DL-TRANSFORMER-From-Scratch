# NumPy

NumPy is the backbone of almost every Machine Learning and Deep Learning library.

Instead of memorizing functions, this folder is about understanding how NumPy actually works and why every ML engineer uses it.

The goal is simple:

- Understand the concepts.
- Write code from scratch.
- Connect every topic to Machine Learning.
- Build intuition for Deep Learning and Transformers.
